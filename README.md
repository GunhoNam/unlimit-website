This is a minimal static "Coming Soon" page ready for GitHub Pages.

## Files
- `index.html` : main page
- `styles.css` : styles
- `script.js` : small countdown + demo form
- `CNAME` : put your custom domain here (replace `yourdomain.com`)

## How to deploy (quick)
1. Create a GitHub repository (public).
2. Copy these files into the repository root.
3. Commit & push to `main` branch.
4. In the repository settings -> Pages, choose branch `main` and folder `/ (root)`.
5. Add your custom domain in Pages settings or place it in `CNAME` file.
6. Add DNS records (see instructions provided separately).

After the DNS propagates (~minutes to hours), enable "Enforce HTTPS" in Pages settings.
